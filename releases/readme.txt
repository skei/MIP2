claptesthost
--------------------

command-line clap host.. i can send a wav or mid file through a plugin, and save the output as wav.. additionally, it can have the plugin load a preset file, assign midi conotrols to parameters, print some info about the plugin, andf do some testing and fuzzing.. works on linux and mac, windows version should be quick and easy, when/if the motivation hits :-)

gain
--------------------

the "helo world" of plugin development..
with a resizable gui

sa_cred
--------------------

note effect.. modifies incomig notes using fibonacci/pisano numbers..

sa_lad
--------------------

ladspa-as-clap wrapper
wraps binary ladspa plugins in /usr/lib/ladspa
linux only
